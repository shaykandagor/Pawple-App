# Pawple – On‑Demand Pet Walking App

Pawple is a mobile application designed to make pet ownership easier, safer, and more flexible. It connects pet owners with trusted walkers, offering instant bookings, real‑time GPS tracking, and a community‑driven experience. Built with modern technologies and user‑centered design, Pawple aims to become a reliable companion for both pet owners and service providers.

## Features

### User Authentication
- Secure account creation and login

### Pet Management
- Add, edit, and delete pet profiles

### On‑Demand Booking
- Instant walk bookings without bulk scheduling

### Real‑Time GPS Tracking
- Live tracking of pets during walks

### Walker Engagement
- Walkers can browse and claim walk requests

### Community Events
- Build connections between pet owners and walkers

### User‑Friendly Interface
- Intuitive navigation and smooth booking flow

---

## Project Overview

Pawple was created to address a clear gap in Helsinki’s pet‑care market: the need for fast, reliable, and affordable walking services. The concept was validated through surveys, interviews, and personal experiences, shaping a solution tailored to real user needs.
